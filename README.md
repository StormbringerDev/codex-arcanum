# Codex Arcanum

The Codex Arcanum is a self-hostable web app inspired by World Anvil that allows for the creation of articles to aid in worldbuilding. You can use this tool for worldbuilding a D&D campaign, another TTRPG campaign, a novel, for fun, whatever you want.

## Installation - Will be updated as development continues

It is recommended to install in a container or external server. However, the server doesn't need to run 24/7 so you can host it on your own PC if you want.

### Prerequisites

- Git
- Node.js
- PostgreSQL (or your database of choice, ensure to adjust code as needed)
- pnpm (recommended), npm, or Yarn

### From Source

1. Clone the repository

```bash
git clone https://github.com/StormbringerDev/codex-arcanum
cd codex-arcanum
```

2. Install dependencies

```bash
pnpm install
```

3. Build the server

```bash
pnpm build
```

4. Run the server

```bash
pnpm start
```
